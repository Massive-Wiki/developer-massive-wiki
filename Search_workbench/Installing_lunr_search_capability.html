<h1>Installing lunr search capability</h1> <h3>2022-07-06/07/10 including lunr search</h3> <p>Steps to incorporate lunr search from developer-massive-wiki to another massive-wiki (running MWB v1.9.0):<br /> 0. merge <code>README.md</code> (done manually)</p> <ol> <li>install node modules: <ul> <li>install node (e.g., macOS: <code>$ brew install node</code>)</li> <li>copy package.json, package-lock.json to <code>.massivewikibuilder</code></li> <li><code>$ npm ci</code> (this installs the node modules into folder <code>node_modules</code>)</li> </ul> </li> <li>install build-index.js in <code>.massivewikibuilder</code></li> <li>in <code>this-wiki-themes/basso</code>: update <code>page.html</code>, <code>_navbar.html</code>, <code>search.html</code></li> <li>merge <code>mwb.py</code> (done manually - a simple copy may do it)</li> <li>update <code>netlify.toml</code> (add <code>--lunr</code> to the mwb build line)</li> </ol>
