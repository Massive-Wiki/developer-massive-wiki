<h1>Installing a Google Font in Basso</h1> <p><em>Last updated 2024-04-10 by Peter Kaminski</em></p> <p><span class="incipient-wikilink">Basso</span> is the current standard theme used for Massive Wikis. You can customize this theme to change how your wiki website looks when built. Here, we will demonstrate how to use a font from Google Fonts, instead of the standard font.</p> <p>To use a font from Google Fonts, follow these steps:</p> <ol> <li>Find a font.</li> <li>Find the embed / import code (CSS).</li> <li>Edit the <code>custom.css</code> file and insert the CSS.</li> <li>Check in your changes.</li> </ol> <p>Assumptions:</p> <ul> <li>Your wiki website is using the Basso them</li> <li>You can edit the <code>custom.css</code> file.</li> <li>You want to use just one font. To use different fonts for different elements of the page, the process is similar, but you'll need to identify the elements you want to change with CSS selectors. That process is typical when using CSS, but is not covered by this page.</li> </ul> <p>Notes:</p> <p>This uses a CSS <code>@import</code> statement so all the changes can be made in <code>custom.css</code>. If you know HTML and CSS, it's probably better to use <code>&lt;link&gt;</code> or <code>&lt;link rel="preload"&gt;</code> in <code>_header.html</code> for faster loading and rendering, while still defining the font family in <code>custom.css</code>. It's a small optimization, though; <code>@import</code> should work fine.</p> <p>Fonts loaded from other web sources can be used as well; Google Fonts is used to demonstrate the concepts required.</p> <h2>Find a font</h2> <p>Go to <a href="https://fonts.google.com/">https://fonts.google.com/</a> and find a font. Navigate to the font's page. For "Roboto", the font page is <a href="https://fonts.google.com/specimen/Roboto">https://fonts.google.com/specimen/Roboto</a>.</p> <p>Click the "Get font" button at the top right. The page should refresh and say "1 font family selected".</p> <h2>Find the embed / import code (CSS)</h2> <p>On the font page, click the "&lt;&gt; Get embed code" button at the top right.</p> <p>Make sure the "Web" tab is chosen, then select the "@import" radio button.</p> <p>Click the "Copy code" link.</p> <p>Paste into a text editor. You should have something like this:</p> <pre><code class="language-html">&lt;style&gt; @import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&amp;display=swap') &lt;/style&gt; </code></pre> <h2>Edit the <code>custom.css</code> file and insert the CSS</h2> <p>Find the <code>custom.css</code> file in your Basso instance. It should be here inside your wiki website:</p> <p><code>.massivewikibuilder/this-wiki-themes/basso/static/mwb-static/css/custom.css</code></p> <p>Edit <code>custom.css</code> with a text editor.</p> <p>From the embed code you found above, copy the line between <code>&lt;style&gt;</code> and <code>&lt;/style&gt;</code>. It should be similar to this:</p> <pre><code class="language-css">@import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&amp;display=swap') </code></pre> <p>Paste it into <code>custom.css</code>.</p> <p>Below that, paste these lines:</p> <pre><code class="language-css">* { font-family: 'Roboto', Arial, sans-serif; font-size: 16px; line-height: 1.5; word-wrap: break-word; } .markdown-body { font-family: 'Roboto', Arial, sans-serif; font-size: 16px; line-height: 1.5; word-wrap: break-word; } </code></pre> <p>Replace <code>'Roboto'</code> with the name of your Google Font.</p> <p>Save <code>custom.css</code>.</p> <h2>Check in your changes</h2> <p>You have now installed the font. Using your normal tools, commit and push your changed file. You can use a commit message like "Installed new Google Font".</p> <p>When your wiki website rebuilds, you should see the new font being used.</p>
