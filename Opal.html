<h1>Opal</h1> <p>One of the <a class="wikilink" href="/developer-massive-wiki/Things_to_Build.html">Things to Build</a>.</p> <p>Also see: <a class="wikilink" href="/developer-massive-wiki/Zircon.html">Zircon</a>, <a class="wikilink" href="/developer-massive-wiki/Obsidian.html">Obsidian</a>.</p> <p>Opal is a desktop (and probably web and mobile) front-end, or client, for Massive Wiki.</p> <p>The idea is to have an easy-to-learn client, which presents an editing pane, a preview pane, and a way to view a list of pages, perhaps in "recently edited" order.</p> <p>It should be able to search all pages, and provide wiki-linking capabilities like Obsidian.</p> <p>It should have Git sync built into it; no separate Git install should be necessary. It should make Git authentication and simple conflict resolution super easy for non-technical users.</p>
