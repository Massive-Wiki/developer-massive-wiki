<h1>Pure.css Side Menu Theme</h1> <p>This is an alpha / in-development theme.</p> <p>It currently lives in the <code>this-wiki-themes/purecss-side-menu</code> directory of the Massive Wiki Developer Wiki.</p> <p>It uses <a href="https://purecss.io/">Pure.css</a>.</p> <p>It was initially created by <a href="mailto:kaminski@istori.com">Peter Kaminski</a>, by lightly adapting the <a href="https://github.com/pure-css/pure/tree/master/site/static/layouts/side-menu">Pure "side menu"</a> layout.</p> <p>Comments, contributions, or forks are welcome.</p> <h2>Wishes and Issues</h2> <p><strong>BUG:</strong> The bounding box for links in the sidebar is too tall, which means it's easy to for a click to go to the wrong link for links that are stacked right on top of one another.</p> <p>Needs a "home page" template (which would need support in MWB).</p> <p>Sidebar is very narrow for typical Sidebar.md content; perhaps it should be wider.</p> <p>Should there be a topbar on every page template for branding?</p> <p>Needs examples of doing color theming / branding. Pure says, "We believe that it's much easier to add new CSS rules than to overwrite existing rules. By adding a few lines of CSS, you can customize Pure's appearance to work with your web project." It then links to <a href="https://purecss.io/extend/">Extend - build on top of Pure</a>.</p> <p>There should be better integration with Pure.css (some of which may require additional features in MWB):</p> <ul> <li>Use the Pure.css menu/navigation classes in the sidebar.</li> <li>Responsive images - "Add the <code>.pure-img</code> class name to an <code>&lt;img&gt;</code> element to make it scale with the viewport."</li> <li>See if we can be clever and use Grids, Forms, Buttons, Tables.</li> </ul> <h2>Resources</h2> <ul> <li><a href="https://purecss.io/">Pure.css home page</a></li> <li><a href="https://www.tutorialspoint.com/purecss/index.htm">Pure.CSS Tutorial - TutorialsPoint</a></li> <li><a href="https://purelog.netlify.app/">Purelog: for Writers</a> by <a href="https://brennanbrown.ca/">Brennan Brown</a> - a Jekyll theme built with Pure, might be a good example for some Pure things.</li> </ul>
