Use tags like python, pypi, and package to reach the right audience.</li> <li>Reddit: Subreddits like r/Python or r/learnpython can be good places to seek advice. There are many experienced developers in these communities who can offer insights and guidance.</li> <li>Python Forums: The official Python forums (discuss.python.org) have a special category for packaging where you can discuss PyPI-related topics and get advice from experienced package maintainers.</li> <li>PyPI's own documentation and community: PyPI provides comprehensive guides and documentation on how to create, distribute, and publish packages. They also have a community and support channels where you can ask for help.</li> <li>Python community channels on IRC or Discord: Channels dedicated to Python development can be valuable for real-time advice and discussion. <ul> <li>https://discord.com/invite/python (Python Discord)</li> <li>https://careerkarma.com/blog/python-discord-server/ (additional Discord servers)</li> </ul> </li> </ul> <p>Places we won't explore for now:</p> <ul> <li>GitHub: If you have a repository for your module, engaging with the GitHub community can be beneficial. You can get feedback through issues or discussions within your repository.</li> <li>Meetups and Conferences: Attending local Python meetups or conferences (like PyCon) can provide you with the opportunity to network with other Python developers and discuss your project for more in-depth feedback.</li> </ul>
