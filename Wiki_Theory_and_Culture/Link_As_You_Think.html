<h1>Link As You Think</h1> <p><em>also see <a class="wikilink" href="/developer-massive-wiki/Linking_Your_Thinking.html">Linking Your Thinking</a></em></p> <p><em>part of <a class="wikilink" href="/developer-massive-wiki/Wiki_Theory_and_Culture/Wiki_Theory_and_Culture.html">Wiki Theory and Culture</a></em></p> <p>from <span class="incipient-wikilink">eekim</span> in 2005:</p> <ul> <li><a href="https://eekim.com/2005/09/the-brilliant-essence-of-wikis/">The Brilliant Essence of Wikis â€“ Eugene Eric Kim</a> (eekim.com)</li> <li><a href="https://web.archive.org/web/20100612154846/http://blueoxen.net/wiki/Link_As_You_Think">Link As You Think</a> (blueoxen.net)</li> </ul> <hr /> <h2>Link As You Think</h2> <p>The core essence of Wikis: the ability to <strong>Link As You Think</strong> by simply typing the page name. What makes this feature beautiful is that you can link to a page that does not already exist.</p> <p>To link to a page on a Wiki, you refer to the page's name. This is different from the default behavior of the web, which usually requires that you track down a URL (usually designed for machines, not people) to create a link.</p> <p>The ability to point to a page that does not currently exist is a powerful capability. <a href="https://web.archive.org/web/20100612154846/http://blueoxen.net/w/index.php?title=Ward_Cunningham&amp;action=edit&amp;redlink=1">Ward Cunningham</a>, who invented the <a href="https://web.archive.org/web/20100612154846/http://blueoxen.net/wiki/WikiWikiWeb">WikiWikiWeb</a>, originally observed this behavior when he was doing software design with index cards. He noticed that engineers would place a card on the table without writing anything on it and talk about it as if it was a real entity, even though it had not yet been named or described.</p> <p><a href="https://web.archive.org/web/20100612154846/http://blueoxen.net/wiki/Link_As_You_Think">Link As You Think</a></p> <hr /> <p><a class="wikilink" href="/developer-massive-wiki/Wiki_Theory_and_Culture/WikiNow.html">WikiNow</a></p>
